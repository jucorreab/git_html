Primeiros passos com o Git Hub

O que é o Git Hub?

O Git Hub é uma plataforma de controle de versão baseado em nuvem e que permitecolaboração para desenvolvimento de software. Com o Git Hub, é possível compartilhar projetos, acompanhar modificações, colaborar com outros desenvolvedores e muito mais.

Passo 1: Acesse o site oficial do Git Hub

Para baixar o Git Hub, acesse o site oficial em https://github.com.

Passo 2: Crie uma conta

Se você ainda não possui uma conta no Git Hub, crie uma clicando no botão "Sign up" ou "Sign in" e siga as instruções para criar uma nova conta.

Passo 3: Baixe o Git Hub Desktop

Após criar uma conta, faça o download do aplicativo Git Hub Desktop para o seu sistema operacional. Você pode encontrar o link para o download na página inicial do site.

Passo 4: Instale o Git Hub Desktop

Execute o arquivo de instalação do Git Hub Desktop que você baixou e siga as instruções para instalá-lo no seu computador.
